What is MULTIPLICATION?

Multiplication is like you add the number before the multiplication symbol as many times as the one after!

For example, if you multiply 9 & 3 it will equal 27 because 9 + 9 + 9 = 27 but if you find that hard you can do 3 + 3 + 3 + 3 + 3 + 3 + 3 + 3 + 3 = 27.
